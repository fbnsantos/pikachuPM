# Política de Privacidade — pikachuPM Todos (extensão Chrome)

**Última atualização:** 20 de maio de 2025

---

## Dados recolhidos

A extensão pikachuPM Todos guarda localmente, através de `chrome.storage.sync`, as seguintes informações introduzidas pelo utilizador:

- URL base da instância pikachuPM
- Token de autenticação da API (Bearer token)
- Preferências de interface (modo compacto, estado do painel lateral)

## Como os dados são utilizados

Estes dados são utilizados exclusivamente para autenticar e comunicar com o servidor pikachuPM configurado pelo próprio utilizador. Nenhum dado é partilhado com terceiros.

## Comunicação de rede

A extensão realiza pedidos HTTP exclusivamente para o servidor configurado pelo utilizador. Não existe qualquer comunicação com servidores externos, serviços de análise ou publicidade.

## Armazenamento

Os dados são guardados via `chrome.storage.sync`, que pode sincronizar entre dispositivos através da conta Google do utilizador, de acordo com a [Política de Privacidade da Google](https://policies.google.com/privacy).

## Eliminação de dados

O utilizador pode eliminar todos os dados guardados pela extensão através de:

1. `chrome://extensions/` → pikachuPM Todos → **Detalhes** → **Limpar dados**
2. Ou desinstalando a extensão

## Contacto

Para questões relacionadas com privacidade, contacta o desenvolvedor através do repositório GitHub da extensão.
