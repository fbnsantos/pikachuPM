⚡ pikachuPM Definições

Ligação à API

O URL raiz da tua instância do pikachuPM, sem barra final.

Encontra o teu token nas definições do pikachuPM ou na tabela user_tokens.

Como obter o token

  1. Inicia sessão no pikachuPM.
  2. Vai a Definições → Conta ou pede ao admin.
  3. Copia o token de API de 64 caracteres.
  4. Cola-o no campo acima.
